Mango provides stunning window effects powered by [SceneFX](https://github.com/wlrfx/scenefx), bringing modern desktop aesthetics to a lightweight Wayland compositor. Customize blur, shadows, corner rounding, and opacity to create your perfect desktop environment.

## Overview

Window effects available in Mango:

- **Blur**: Background blur for windows and layer surfaces
- **Shadows**: Customizable drop shadows with blur and positioning
- **Corner Radius**: Rounded corners for modern aesthetics
- **Opacity**: Adjust focused and unfocused window transparency

<Info>
All window effects are optional and can be disabled individually for maximum performance.
</Info>

## Blur Effects

Background blur creates translucent windows with blurred content behind them.

### Blur Configuration

```conf
# Enable blur for windows
blur=0

# Enable blur for layer surfaces (bars, launchers)
blur_layer=0

# Use optimized blur algorithm (better performance)
blur_optimized=1

# Blur algorithm parameters
blur_params_num_passes=2
blur_params_radius=5
blur_params_noise=0.02
blur_params_brightness=0.9
blur_params_contrast=0.9
blur_params_saturation=1.2
```

### Blur Parameters

<ParamField path="blur" type="boolean" default="0">
  Enable blur effect for windows (0=disabled, 1=enabled)
</ParamField>

<ParamField path="blur_layer" type="boolean" default="0">
  Enable blur effect for layer surfaces like bars and menus
</ParamField>

<ParamField path="blur_optimized" type="boolean" default="1">
  Use optimized blur algorithm for better performance
</ParamField>

<ParamField path="blur_params_num_passes" type="integer" default="2">
  Number of blur passes. More passes = smoother blur but slower performance
</ParamField>

<ParamField path="blur_params_radius" type="integer" default="5">
  Blur radius in pixels. Higher values = more blur
</ParamField>

<ParamField path="blur_params_noise" type="float" default="0.02">
  Amount of noise to add (reduces banding artifacts)
</ParamField>

<ParamField path="blur_params_brightness" type="float" default="0.9">
  Brightness adjustment for blurred content (0.0-2.0)
</ParamField>

<ParamField path="blur_params_contrast" type="float" default="0.9">
  Contrast adjustment for blurred content (0.0-2.0)
</ParamField>

<ParamField path="blur_params_saturation" type="float" default="1.2">
  Saturation adjustment for blurred content (0.0-2.0)
</ParamField>

## Shadow Effects

Add depth to your desktop with customizable drop shadows.

### Shadow Configuration

```conf
# Enable shadows for windows
shadows=0

# Enable shadows for layer surfaces
layer_shadows=0

# Only show shadows on floating windows
shadow_only_floating=1

# Shadow size (spread)
shadows_size=10

# Shadow blur amount
shadows_blur=15

# Shadow offset X (pixels)
shadows_position_x=0

# Shadow offset Y (pixels)
shadows_position_y=0

# Shadow color (RGBA hex)
shadowscolor=0x000000ff
```

### Shadow Parameters

<ParamField path="shadows" type="boolean" default="0">
  Enable drop shadows for windows
</ParamField>

<ParamField path="layer_shadows" type="boolean" default="0">
  Enable shadows for layer surfaces
</ParamField>

<ParamField path="shadow_only_floating" type="boolean" default="1">
  Restrict shadows to floating windows only (0=all windows, 1=floating only)
</ParamField>

<ParamField path="shadows_size" type="integer" default="10">
  Shadow spread distance in pixels
</ParamField>

<ParamField path="shadows_blur" type="integer" default="15">
  Shadow blur radius in pixels
</ParamField>

<ParamField path="shadows_position_x" type="integer" default="0">
  Shadow horizontal offset (negative = left, positive = right)
</ParamField>

<ParamField path="shadows_position_y" type="integer" default="0">
  Shadow vertical offset (negative = up, positive = down)
</ParamField>

<ParamField path="shadowscolor" type="hex" default="0x000000ff">
  Shadow color in RGBA hex format (0xRRGGBBAA)
</ParamField>

### Shadow Examples

  ```conf Soft Shadow
  shadows=1
  ```

</CodeGroup>

## Corner Radius

Round window corners for a modern, polished look.

### Corner Radius Configuration

```conf
# Corner radius in pixels
border_radius=6

# Disable radius when only one window
no_radius_when_single=0
```

<ParamField path="border_radius" type="integer" default="6">
  Corner radius in pixels (0=sharp corners)
</ParamField>

<ParamField path="no_radius_when_single" type="boolean" default="0">
  Disable corner radius when only one window is visible (0=always round, 1=sharp when single)
</ParamField>

### Radius Suggestions

- **Subtle**: `border_radius=4` - Slightly softened corners
- **Modern**: `border_radius=8` - Noticeable but tasteful
- **Bold**: `border_radius=12` - Prominent rounded corners
- **Extreme**: `border_radius=20` - Very rounded, macOS-like

<Info>
Corner radius applies to window content and borders, creating a cohesive appearance.
</Info>

## Opacity

Adjust window transparency for focused and unfocused windows.

### Opacity Configuration

```conf
# Focused window opacity (0.0-1.0)
focused_opacity=1.0

# Unfocused window opacity (0.0-1.0)
unfocused_opacity=1.0
```

<ParamField path="focused_opacity" type="float" default="1.0">
  Opacity of the focused window (1.0=fully opaque, 0.0=fully transparent)
</ParamField>

<ParamField path="unfocused_opacity" type="float" default="1.0">
  Opacity of unfocused windows (1.0=fully opaque, 0.0=fully transparent)
</ParamField>

## Performance Considerations

Window effects impact GPU performance. Optimize for your hardware:

### High-End GPU

```conf
# Enable all effects
blur=1
blur_layer=1
blur_params_num_passes=3
blur_params_radius=8
shadows=1
layer_shadows=1
shadows_blur=25
border_radius=8
```

### Mid-Range GPU

```conf
# Balanced effects
blur=1
blur_layer=0
blur_optimized=1
blur_params_num_passes=2
blur_params_radius=5
shadows=1
shadow_only_floating=1
shadows_blur=15
border_radius=6
```

### Low-End GPU / Battery Saving

```conf
# Minimal effects
blur=0
blur_layer=0
shadows=0
layer_shadows=0
border_radius=4
focused_opacity=1.0
unfocused_opacity=0.95
```

<Warning>
Blur is the most expensive effect. If experiencing performance issues, disable blur first.
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Effects not appearing">
    Ensure SceneFX is installed correctly. Mango requires SceneFX for window effects. Check installation:
    ```bash
    ldconfig -p | grep scenefx
    ```
  </Accordion>
  
  <Accordion title="Performance issues with blur">
    Reduce blur quality:
    - Set `blur_optimized=1`
    - Reduce `blur_params_num_passes` to 1
    - Reduce `blur_params_radius` to 3
    - Disable `blur_layer` for layer surfaces
  </Accordion>
  
  <Accordion title="Shadows appear clipped">
    Increase `shadows_size` and reduce `shadows_blur` for more contained shadows.
  </Accordion>
  
  <Accordion title="Corner radius looks wrong">
    Some applications don't respect corner radius hints. This is an application limitation, not a compositor issue.
  </Accordion>
</AccordionGroup>
